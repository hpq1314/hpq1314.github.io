阴阳师又要更新啦

发表于 2017-06-22   |     |   阅读次数

阴阳师烟月春和新版本将于1月22日正式更新，除了新式神新皮肤外，还有许多的新玩法。下面就让为大家汇总下阴阳师烟月春和新版本更新汇总，希望对大家有所帮助。


一、新式神
包括SSR辉夜姬和SR烟烟罗，两个高颜值美妞。

式神图鉴>>>辉夜姬



辉夜姬（小姐姐跟管狐什么关系？）


式神图鉴>>>烟烟罗



烟烟罗（女版食发鬼？）

二、现世召唤
类似画符召唤的全新方式，玩家可以手绘或打印现世召唤阵,将召唤阵平置于设备镜头前,就可以使用现世符咒召唤式神了。

阴阳师现世符咒召唤 召唤式神新姿势（提供召唤阵）
