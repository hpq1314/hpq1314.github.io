吃喝在汕头(一)

发表于 2017-01-18   |     |   阅读次数

作为汕头的孩子，觉得生活在汕头是超级幸福的！为什么呢？
因为这里有好多地道的潮汕美食啊，来汕头逛，必不可少的就是 吃吃吃。
下面，我就带大家看看汕头有哪些好吃的吧

说到汕头，很多外地人的第一印象就是牛肉丸，可见牛肉丸已经成为了汕头的标志性美食。汕头牛肉丸又分为牛肉丸和牛筋丸，牛肉丸采用纯鲜肉打制，肉质细嫩，口感松脆；牛筋丸是在牛肉里加进嫩筋，使之味道更香，弹性更强，口感爽脆、柔嫩，且丸中带汤，口嚼美汁四溅。



牛肉丸



牛筋丸

随着来汕的外地人希望尝到那种“掉地上能弹的老高”的牛肉丸，牛肉丸的制作要求也越来越高，现普遍好吃的牛肉丸都是经过手打而成的。食时用原汤和牛肉丸下锅煮至初沸（煮时水不能太沸，否则牛肉丸不爽滑），加入适量味精、芝麻油、胡椒粉和芹菜粒，配上沙茶酱或辣椒酱佐食。吃牛肉丸有不少人蘸辣椒酱，不过还是蘸本地特制沙茶酱最为地道。牛肉丸还可烤着吃，烧烤时把牛肉丸剖成两半，抹上酱料和蜂蜜烤熟即可大嚼。
汕头牛肉丸比较好吃的店有海记、福合埕等等。

无米粿也是汕头著名美食之一，又称韭菜粿。无米粿的皮是番薯粉做的。无米粿的馅多用蔬菜杂粮来做。咸的大多是韭菜、马铃薯、芋头或竹笋切成小粒，加沙茶或咖喱及盐、味精少量炒熟而成。吃之前呢，大多数人会选择用“煎”来锦上添花。

无米粿与潮汕特制的辣椒酱是最佳搭配。辣椒酱不甜也不是很辣，有点咸。夹一个无米粿蘸点辣椒酱，放入嘴里轻轻一咬，你会觉得外皮很酥脆，再咬深一点，你会觉得韭菜的菜汁及辣酱的浆汁混合在一起，满嘴都是那种香气。由于特别的烫嘴，以至你不得不张开口呼出腾腾的热气，又迫不及待狠狠咀嚼几口，意犹未尽。

无米粿比较好吃的店有福合沟、李记等等。

鲎粿，这个是汕头潮阳区棉城、海门特有的汉族传统小食，最传统的做法是取鲎肉与米浆混合，加入猪肉末香菇鹧鸪蛋等食材一同炊制，最后再经油浸高温烹制，捞起来后淋上特制沙茶酱，极具潮阳乡土味。

煎饺+草莓冰，这是汕头老市区一道特色的美食搭配。
煎饺也就是油煎饺子，老市区卖煎饺比较出名的就是汕头第十二中学旁的煎饺摊子，每次路过都觉得”香气逼人“。


草莓冰，是往冰块中加入草莓肉，倒入调料，再用榨汁机榨成的。草莓冰上还可以加上酸奶、炼奶、草莓肉、雪糕等。汕头草莓冰比较出名的店有亚强果汁冰和十二中草莓冰。草莓是冬天当季，尽管冬天很冷，但还是有很多人慕名前去十二中旁排长队买一杯鲜榨的草莓冰。

吃着盘子热热的煎饺，再吸一口草莓冰，说不出的幸福。
