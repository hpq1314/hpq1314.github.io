吃喝在汕头(二)

发表于 2017-01-23   |     |   阅读次数

上次给大家介绍了牛肉丸、无米粿、鲎粿等汕头美食，看完的你们是不是已经垂涎三尺了呢？那今天就给大家介绍另外几样潮汕小吃，别流口水噢

蚝烙，潮汕地方特色小吃，台湾称其为“蚵仔煎”。

蚝烙的传统做法是，先热鼎落油，放下葱花炒出香味，再把生粉水均匀地倒下，煎成圆饼形，约有一公分厚，至生粉水刚熟成形，即把蛋浆均匀淋上，再在上面放蚝(用粉水上浆)、腊肉丁等，略煎，用锅铲切成四角，从鼎边再落朥，翻过来继续煎至外香脆，内嫩滑。”蚝烙”的酱碟一般是鱼露。

汕头比较有名的蚝烙店有榕香蚝烙和飘香小食店


榕香蚝烙店


飘香小食店


水粿，也叫粿仔。

将米浆倒入摆好的模具里，然後再放到锅炉上猛火蒸熟，熟之后待冷却，取出蒸熟的米浆就会呈碗状。再蒸热，一个一个摆好放在盘子里，然后放上菜脯粒，浇上特制甜酱，一道潮汕美食就诞生啦。



肠粉，也是用米浆做成的一道美食。

传统做法是在大网筛子上铺一块白布，将磨好的米浆浇在白布上，隔水蒸熟成粉皮，再在粉皮上放上馅料（鸡蛋、肉片、鱼片、虾仁等），卷成猪肠形。

潮汕的肠粉和广式的布拉肠吃起来可不一样噢，广式布拉肠的粉皮透明且有弹性，对外观的要求比潮汕肠粉高，精致，味道清；而潮汕肠粉馅料多、配料多、酱汁味道多样。
汕头肠粉推荐店有三身人金新肠粉、芙蓉肠粉、十二中肠粉等。






豆花，是将干黄豆磨成的豆浆烧开后加入到石膏粉做成的稀软的的固体小吃。

一般吃法是在豆花上倒上一层白砂糖。

汕头做豆花比较出名的店是广场老牌豆花店


粿汁，潮汕地区大众化的传统民间小食，属于潮菜。粿汁皮是潮汕粿汁的主料，其中添加一些配料和淋上浅棕色的卤汁使得粿汁更加的美味可口。

配料除了香腾腾的葱珠粒是必备的之外，卤蛋、豆腐、猪肠、猪肉、猪杂等也是不可或缺的。

粿汁推荐店有老姿娘粿汁店和广场潮州粿汁店。

老姿娘粿汁店


广场潮州粿汁店



本次《吃喝在汕头》美食介绍系列就到这里，还有更多美食等着你亲身来了解，相信看完一二篇《吃喝在汕头》的你们已经是迫不及待想来汕头一饱口福了哈哈。

小编我觉得汕头就是一个你来了，吃了，就不想走的城市啊。
这片土地滋养着很多很有名的小吃，但是萝卜青菜，各有所爱，只要是你觉得好吃的，它就是你心中的美食。
